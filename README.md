# DSA Practice Archive

A collection of Data Structures & Algorithms (DSA) solutions, LeetCode problems, coding patterns, and Java-based problem-solving exercises created during my interview preparation journey.

This repository serves as a structured archive for practicing algorithms, improving problem-solving skills, and preparing for software engineering interviews.

## ▶️ Getting Started

### Clone the Repository

```bash
git clone https://github.com/gautamthapa/dsa-practice-archive.git
```

### Navigate to the Project

```bash
cd dsa-practice-archive
```

### Build the Project

```bash
mvn clean install
```

### Run the Application

```bash
mvn exec:java
```

Or execute individual Java classes directly from your IDE.
